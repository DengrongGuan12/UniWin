get /v1/styles

获取款式列表

  • Request
  • Response

Query Parameters

  • page: required (integer)

    页数从0开始

    Example:

    1
  • num: required (integer)

    每页多少项

    Example:

    10
  • operation: required (string)

    Example:

    SEARCH/NORMAL
  • key: (string)

    如果operation是SEARCH,该值就是要搜索的关键词

    Example:

    201674
  • field: (string)

    要搜索的域,域之间用|隔开

    Example:

    CODE|NAME

HTTP status code 200

Body

Type: application/json

Example:

{
  "result":0失败1成功,
  "data":[{
  "code":款号,
  "imgUrl": 图片地址,
  "description": 描述,
  "pass":true或false
  },{},...],
  "error":{
    "code":错误码,"message":错误信息
  }
}

get /v1/materials

获取物料信息

    post /v1/materials

    • Request
    • Response

    Query Parameters

    • password: required (string)

      使用token加密

      Example:

      12345
    • userName: required (string)

      必须由字母数字组成

      Example:

      abcd
    • sex: required (string)

      Example:

      0(男),1(女)
    • birthday: required (string)

      Example:

      2015-11-09,注意格式
    • schoolId: required (number)

      Example:

      1
    • departmentId: required (number)

      Example:

      1

    HTTP status code 200

    Body

    Type: application/json

    Example:

    {
  "data":{
    "result":0(表示失败,用户名重复),非0(表示userId)
  },
  "success": true,
  "status": 200
}
