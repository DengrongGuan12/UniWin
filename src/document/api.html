get /v1/styles

获取款式列表

  • Request
  • Response

Query Parameters

  • page: required (integer)

    页数从0开始

    Example:

    1
  • num: required (integer)

    每页多少项

    Example:

    10
  • operation: required (string)

    Example:

    SEARCH/NORMAL
  • key: (string)

    如果operation是SEARCH,该值就是要搜索的关键词

    Example:

    201674
  • field: (string)

    要搜索的域,域之间用|隔开

    Example:

    code|description

HTTP status code 200

Body

Type: application/json

Example:

{
  "result":0失败1成功,
  "data":{
      "count":100,
      "list":[{
          "code":款号,
          "imgUrl": 图片地址,
          "description": 描述,
          "passed":true或false,
          "createTimeStr":创建时间
          },{},...]
  },
  "error":{
    "code":错误码,"message":错误信息
  }
}
